# Süt Pompası Kiralama Sivas

Hastane tipi süt sağma pompası (Medela, Spectra, Ameda) kiralama tanıtım sitesi. Türkiye geneli kargo ve kiralama hizmeti — Sivas merkezli.

## İçerik

- **`index.html`** — Tek dosyalık, bağımsız (standalone) statik site. Harici bir build adımı gerektirmez.
- Yazı tipleri Google Fonts'tan (Playfair Display + Mulish), ürün görselleri sitenin kendi WordPress medyasından yüklenir.
- Tamamen **mobil uyumlu** (responsive) — 380px / 640px / 960px kırılım noktaları.
